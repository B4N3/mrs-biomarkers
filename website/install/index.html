Installing the BRP Client Software

  • Download above zip file.
  • Extract contents of zip file to C:\MedUser\MriCustomer\seq folder.
  • Use notepad to edit C:\MedUser\MriCustomer\seq\brp\brp-config.bat
  • Confirm the software path.
  • Set the encryption password for data.
  • Set your site login user and password for upload (future).
  • Communicate the settings to the BRP central site contact.

Using the BRP Client Software

  • Run the MP-RAGE sequence.
  • Save the DICOM data in C:\MedUser\MriCustomer\seq\brp\data in a folder (e.g. folder1).
  • CTL-ESC and open a command window by running cmd.exe.
  • cd C:\MedUser\MriCustomer\seq\brp
  • brp-send-receive.bat data\folder1
  • Wait 45 seconds. The status will change from ready to received and then to complete.
  • Run the MRS sequence. It will use the voxel results in brp\inbox\coordinates.ini.

BRP relay software

You only need this if you can't reach http://brp.cmrr.umn.edu or http://brp.cmrr.umn.edu:81 directly from your scanner!

Download BRP relay software
Download BRP relay website

Installing the BRP Relay software

  • Download the brprelay.tgz and extract under /opt. You now have /opt/brp.
  • Modify /opt/brp/site.passwords to include the user id and password for the main BRP website.
  • Modify /opt/brp/relay for your website. Run /opt/brp/relay at system start to make the relay function. It will check for new uploads and downloads.
  • Make sure your website is functioning already. This was developed using apache on linux.
  • Download the brprelay-website.tgz and extract under your website html directory /website. You now have /website/brprelay.
  • Add a .htaccess file and set a password for your /website/brprelay directory to protect it. Check this with a web browser.
  • Use this user id and password on your brp-config.bat file on the scanner. Change the URL there to point at the relay (http://yourserver/brprelay/api.php)
  • At this point you should be able to run the scanner, send to the relay, and have the relay forward the data. The relay will watch for a result and put it in place for the scanner in the relay website.
